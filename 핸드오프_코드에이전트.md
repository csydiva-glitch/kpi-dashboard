## 1. 프로젝트 컨텍스트

### 1.1 비즈니스 배경

한솔제지 경영혁신팀은 중기전략 수립 시 도출한 KI(Key Initiative)별 지표의 실적 및 달성률을 매월 점검하고 경영진에게 보고한다. 기존엔 KI 관리시스템에서 엑셀 추출 → 수작업 PPT 작성에 **6시간/월**이 소요됐다. 이 대시보드는 차질 KPI를 즉시 식별 가능하도록 시각화하고 PPT 자동 생성으로 보고 작업을 단축한다.

### 1.2 사용자 인터뷰 결과 (확정 사양)

| 항목 | 결정 사항 |
|---|---|
| 신호등 기준 | 🟢 ≥90% / 🟡 70~90% / 🔴 <70% (단일 임계, 초과달성 별도 구분 없음) |
| 신호등 상·하한 | 상한 200% 캡, 음수는 0% 보정 |

### 2.4 데이터의 특성 (주의 포인트)

1. **음수 계획**: 일부 KPI는 적자 목표(예: -50억 손실 계획)를 가짐 — 환경 운영, Eco패키징 등
2. **음수 실적**: 흑자 계획에 적자 실적이 나는 케이스 존재 — 인쇄 (유통) 경상이익 등
3. **누계 실적 0 행**: 1월 26건 → 12월 3건 (시점이 안 온 KPI 또는 0 기록)
4. **달성률 컬럼이 이미 산출되어 있음**: 별도 재계산보다 이 값을 사용하는 것이 안전 (단, 적자 KPI 예외)

### 2.5 정제 코드 (Python)

```python
import pandas as pd
import json, math

tables = pd.read_html('25년_실적.xls', encoding='utf-8')
df = tables[1].copy()  # 두 번째 테이블이 실제 데이터
df.columns = [c.strip() for c in df.columns]
df = df.drop(columns=['삭제여부'])
df['분류'] = df['분류'].str.replace('/', '').str.strip()

# 숫자 컬럼 변환
for c in ['연간 계획','당월 누계 계획','당월 누계 실적','달성율(%)','진척율(%)']:
    df[c] = pd.to_numeric(df[c], errors='coerce')

df = df.rename(columns={
    'KPI/재무목표': '지표명',
    '지표': '구분',
    '핵심KPI': '차별화',
})
df['세부실행과제'] = df['세부실행과제'].fillna('')
df['KPI_ID'] = df['부문'] + '|' + df['KI'].str[:50] + '|' + df['지표명']

# NaN → None
def clean(r):
    return {k: (None if isinstance(v, float) and math.isnan(v) else v) for k, v in r.items()}
records = [clean(r) for r in df.to_dict('records')]

with open('data.json', 'w', encoding='utf-8') as f:
    json.dump(records, f, ensure_ascii=False)
```

## 3. 핵심 비즈니스 룰 (반드시 준수)

### 3.1 신호등 기준

```javascript
const TL_GREEN = 90;   // 정상 임계
const TL_YELLOW = 70;  // 주의 임계
const CAP = 200;       // 달성률 상한
const WARN_DROP = 20;  // 전월 대비 급락 임계 (%p)

function tl(ach){
  if(ach === null) return 'gray';
  if(ach >= TL_GREEN) return 'green';   // ≥ 90%
  if(ach >= TL_YELLOW) return 'yellow'; // 70 ≤ x < 90
  return 'red';                          // < 70%
}
```

### 3.2 달성률 산출 로직 (가장 중요)

**우선순위 적용:**

```javascript
function calcAch(plan, actual, excelAch){
  // 1) 계획=0/null → N/A
  if(plan === null || plan === undefined || isNaN(plan) || plan === 0) return null;

  let v;
  if(plan < 0 && actual !== null && actual !== undefined && !isNaN(actual) && actual < 0){
    // 2) 적자 KPI (계획<0 & 실적<0) → 절대값 역산
    //    적자 폭이 작을수록(=실적의 |값|이 작을수록) 달성률↑
    v = (Math.abs(plan) / Math.abs(actual)) * 100;
  } else {
    // 3) 일반 케이스 → 엑셀 '달성율(%)' 컬럼값 그대로 사용
    v = excelAch;
    if(v === null || v === undefined || isNaN(v)){
      // 엑셀에 값 없으면 직접 계산
      if(actual === null || actual === undefined || isNaN(actual)) return null;
      v = (actual/plan) * 100;
    }
  }

  // 4) 보정
  if(v < 0) v = 0;       // 음수 → 0%
  if(v > CAP) v = CAP;   // 200% 초과 → 200%
  return v;
}
```

**중요한 의미:**
- 적자 KPI는 "적자 축소 = 좋은 결과"로 해석됨 → 절대값 역산
  - 예: 계획 -50억, 실적 -40억 (적자 축소) → 50/40 = 125% → 🟢
  - 예: 계획 -50억, 실적 -100억 (적자 확대) → 50/100 = 50% → 🔴
- 일반 KPI는 엑셀의 `달성율(%)` 컬럼값을 신뢰 (별도 계산 안 함)
- 부문/카테고리 합산값은 집계 후 직접 계산 (`calcAch(합산계획, 합산실적)`)

### 3.3 N/A 처리

| 케이스 | 처리 |
|---|---|
| 누계 계획 = 0 | ⚪ N/A, 산출 제외 |
| 누계 계획 = null/NaN | ⚪ N/A, 산출 제외 |
| 실적 = null (계획 양수) | ⚪ N/A |
| 실적 = 0 (계획 양수) | 0%, 🔴 차질 |

### 3.4 단월 차분 산식 (재무목표 차트만)

엑셀에는 단월값이 없으므로:
```
단월값(m) = 누계값(m) - 누계값(m-1)
단월값(1월) = 누계값(1월)
```
음수 차분도 그대로 표시 (보정하지 않음).

### 3.5 전월 대비 변화

```javascript
delta = 당월 달성률 - 전월 달성률
warn = (delta <= -WARN_DROP)  // -20%p 이상 하락
```
1월은 전월 데이터가 없으므로 변화 표시 안 함.

## 9. 기능별 동작 명세 (Functional Spec)

### 9.1 월 슬라이더 (FR-C1)

- 범위: 0~12, 초기값 0
- 0 = 미선택 (빈 상태 화면 표시)
- 1~12 선택 시 즉시 전체 화면 갱신
- 사용자가 0으로 돌리면 다시 빈 상태로 복귀

### 9.2 칩 필터 (FR-C2~C5)

- 클릭 시 active 토글
- 모든 active 칩의 교집합으로 행 필터링
- 전체 비활성화하면 빈 결과

### 9.3 부문 카드 드릴다운

- 클릭 시 `selectedDiv` 설정 → 상세 테이블에 해당 부문만 표시
- 같은 카드 재클릭 → 선택 해제 (전체 보기)
- "전체 보기" 버튼 → `resetSelection()` 호출

### 9.4 상세 테이블

- 정렬 가능 컬럼: 부문, 핵심 KPI, 누계 계획, 누계 실적, 달성률
- 기본 정렬: 달성률 오름차순 (차질 KPI 상단)
- 검색: 핵심 KPI 이름 부분 일치 (대소문자 무관)
- 메모: 인라인 입력, blur 시 localStorage 저장, 입력값 있으면 노란 배경

### 9.5 재무목표 차트

- **누계 탭**: 라인 차트, 계획(회색 점선) vs 실적(다크 실선)
- **단월 탭**: 막대 차트(계획/실적) + 라인(달성률), 콤보형
- 차트 인스턴스는 매번 destroy 후 재생성

### 9.6 PPT 다운로드

- 월 미선택 시 alert로 차단
- 파일명: `KPI_대시보드_{월}월_{YYYY-MM-DD}.pptx`
- 메모는 Worst Top 10 슬라이드 4번째 컬럼에 자동 삽입

### 9.7 인쇄

- `window.print()` 호출
- `@media print` CSS로 자동 최적화 (A4 세로, 14mm 마진)
- 컨트롤·메모 입력·버튼 숨김, 색상은 그대로 출력

## 부록 A. 핵심 코드 발췌 (calcAch 함수 전체)

```javascript
const TL_GREEN = 90, TL_YELLOW = 70, CAP = 200, WARN_DROP = 20;

function calcAch(plan, actual, excelAch){
  if(plan === null || plan === undefined || isNaN(plan) || plan === 0) return null;
  let v;
  if(plan < 0 && actual !== null && actual !== undefined && !isNaN(actual) && actual < 0){
    v = (Math.abs(plan) / Math.abs(actual)) * 100;
  } else {
    v = excelAch;
    if(v === null || v === undefined || isNaN(v)){
      if(actual === null || actual === undefined || isNaN(actual)) return null;
      v = (actual/plan) * 100;
    }
  }
  if(v < 0) v = 0;
  if(v > CAP) v = CAP;
  return v;
}

function tl(ach){
  if(ach === null) return 'gray';
  if(ach >= TL_GREEN) return 'green';
  if(ach >= TL_YELLOW) return 'yellow';
  return 'red';
}
```
